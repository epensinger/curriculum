Add the Form

  1. On getbootstrap.com navigate to components > form
  2. Add the form <form></form>
  3. Add the form group div <div class="form-group"></div>
  4. Paste in the email address group <label for="exampleInputEmail1">Email address</label> <input type="email" class="form-control" id="exampleInputEmail1" aria-describedby="emailHelp"> <small id="emailHelp" class="form-text text-muted">We'll never share your email with anyone else.</small>
  5. Adjust the email address group <label for="user_email">Email address</label> <input type="email" class="form-control" id="user_email" aria-describedby="emailHelp"> <small id="emailHelp" class="form-text text-muted">We'll never share your email with anyone else.</small>
  6. Add a button <button type="submit" class="btn btn-primary">Submit</button>
  7. Play with button options (bootstrap's color system) class="btn btn-primary" class="btn btn-secondary" class="btn btn-success" class="btn btn-danger" class="btn btn-warning" class="btn btn-info" class="btn btn-light" class="btn btn-dark"
  8. Add a file input <form> <div class="form-group"> <label for="profile_upload">Profile Uploadt</label> <input type="file" class="form-control-file" id="profile_upload"> </div> </form>
  9. Add a check box <div class="form-check"> <input class="form-check-input" type="checkbox" value="" id="signUp"> <label class="form-check-label" for="signUp"> Join the Mailing List </label> </div>
  10. Get the user's password <div class="form-group"> <label for="user_password">Password</label> <input type="password" class="form-control" id="user_password"> </div>

    11. Add a Navbar while we are at it.

  11. Add the navbar wrapper <nav class="navbar navbar-light bg-light"></nav>
  12. Add the menu items <div class="navbar-nav"> <a class="nav-link" href="#">Home <span class="sr-only">(current)</span></a> <a class="nav-link" href="#">Services</a> <a class="nav-link" href="#">Pricing</a> <a class="nav-link" href="#">Meet The Team</a> </div>
  13. Add a Dropdown <a class="nav-link dropdown-toggle" href="#" id="navbarDropdown" role="button" data-toggle="dropdown" aria-haspopup="true" aria-expanded="false"> Services</a> <div class="dropdown-menu" aria-labelledby="navbarDropdown"> <a class="dropdown-item" href="#">Action</a> <a class="dropdown-item" href="#">Another action</a> </div>
  14. Make it go away on narrow screens - add this around the "navbar nav" to make whatever is inside this div collapse then the navbar collapses <div class="collapse navbar-collapse" id="navbarNav">
  15. add navbar-expand-lg to the nav element to make it expand when it hits the large breakpoint.
  16. breakpoints are listed in the layout overview page in the bootstrap docs.
  17. Add a toggler to expand the menu on smaller screens. <button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#navbarNav" aria-controls="navbarSupportedContent" aria-expanded="false" aria-label="Toggle navigation"> <span class="navbar-toggler-icon"></span> </button>
    • bootstrap is using its grid system to stack the items in the nav on a small screen and have them unstacked on a wider screen. You will learn how to use that grid soon.
    • the data target matches the ID of the "navbarNav" so it will open the navbarNav
